온라인 스톱워치 — 랩·메모·CSV 내보내기

랩마다 이름·메모를 남길 수 있는 스톱워치. 달리기 기록, 팀 릴레이, 발표 리허설, 인터벌 훈련까지 — 세션 전체를 저장하고 CSV로 내보내 엑셀·구글 스프레드시트에서 분석할 수 있습니다. 가입 없이 브라우저에서 무료로 바로 사용하세요.

▶스톱워치 시작하기00:00.00에서 즉시 시작

ReadySetTimer 스톱워치의 강점

  • 1/100초 정밀도 — performance.now() 기반, mm:ss.cs 표시
  • 랩 + 메모 — 각 랩 옆에 이름·내용을 바로 입력 (예: "Alice", "3세트")
  • 최고/최악 랩 자동 하이라이트 — 초록/빨강 색상으로 한눈에
  • 세션 저장 — 제목·노트와 함께 최대 50건까지 브라우저에 영구 보관
  • CSV 내보내기 — UTF-8 BOM 포함으로 엑셀에서 한글 안 깨짐
  • '모두 CSV로' — 여러 세션을 한 파일로 합쳐 다운로드
  • 키보드 단축키 — Space 시작/정지, L 랩, S 저장, R 초기화

이런 사람에게 좋아요

  • 러너 — 인터벌 훈련, 팀 릴레이, 기록 추적
  • 강사·트레이너 — 수강생별 세트 시간 측정 후 CSV로 공유
  • 발표·MC — 슬라이드 구간별 타이밍 기록
  • 요리사 — 레시피 단계별 소요 시간 기록·최적화
  • 엔지니어 — 작업 단계별 시간 측정, 생산성 분석

사용법 (5단계)

  1. 위의 "스톱워치 시작하기" 또는 메인 페이지 스톱워치 탭 클릭
  2. Space 키로 스톱워치 시작 → 00:00.00부터 측정 시작
  3. 구간마다 L 키 또는 "랩" 버튼 → 행이 추가되고 메모 입력 칸 생성
  4. 메모 칸에 이름·내용 입력 (탭 이동 or 클릭)
  5. 완료 후 "기록 저장" → 세션 제목·노트 입력 → 저장 → 필요 시 CSV 내보내기

자주 묻는 질문

백그라운드 탭에서도 정확한가요?

네. 다른 탭으로 이동하거나 브라우저를 최소화해도 내부 시계는 그대로 흐릅니다. 복귀 시 정확한 경과 시간이 표시됩니다.

50건 넘으면 어떻게 되나요?

오래된 기록부터 자동 삭제됩니다. 중요한 기록은 그때그때 CSV로 내보내 보관하세요.

다른 기기에서도 기록을 볼 수 있나요?

기록은 브라우저 localStorage에만 저장되므로 기기 간 동기화는 되지 않습니다. CSV 내보내기 후 Dropbox·Google Drive에 올리면 어디서든 볼 수 있습니다.
