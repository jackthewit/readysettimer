免费在线计时器 · 番茄钟 · 秒表

ReadySetTimer 是一个免费的在线计时器。无需注册、无需下载,直接在浏览器中使用。具有演示级的大号数字、自动循环的番茄钟,以及支持每圈备注和 CSV 导出的秒表。

▶开始计时器25:00 倒计时 · 按空格键或点击开始

特色功能

  • 演讲级大号数字和圆形进度环,礼堂后排也清晰可见
  • 背景色随剩余时间变化(青 → 橙 → 红)
  • 最后 6 秒全屏大数字倒计时
  • 番茄钟自动循环 — 25/5/15 分钟,可自定义
  • 秒表支持每圈备注(如"张三"、"第 3 组")并可导出 CSV
  • 全屏专注模式 — 按 F 键
  • 无账号、无服务器、无追踪 — 数据仅存于您的浏览器

适合使用场景

  • 演讲者 / 教师 — 演示倒计时、课时管理、辩论计时
  • 学生 — 番茄工作法、考试模拟
  • 厨师 — 3 分钟、5 分钟等快速提醒
  • 跑者 / 教练 — 间歇训练,团队接力,CSV 记录
  • 远程会议 — "每人 3 分钟"的轮流发言

常见问题

番茄钟的时长可以调整吗?

可以。在设置 > 番茄钟中调整专注、短休息、长休息和长休息周期(例如 50/10、20/5)。

如何书签一个特定时长?

使用 URL 参数:rstimer.kr/?t=5m、?t=25m、?t=1h30m 等,打开时自动预设。

记录能同步到其他设备吗?

数据仅存于浏览器 localStorage,不跨设备同步。导出为 CSV 后可上传到云端查看。
