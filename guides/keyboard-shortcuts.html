ReadySetTimer 키보드 단축키 완전 정리

최종 업데이트: 2026-04-24 · 약 5분 읽기

발표 중에 마우스 커서가 화면에 나타나면 청중의 시선이 분산됩니다. 강의·온라인 미팅·웹 시연에서도 마찬가지예요. 그래서 ReadySetTimer는 모든 핵심 기능을 키보드만으로 통제할 수 있게 설계되었습니다. 이 가이드는 모든 단축키를 한 페이지에 정리해 두었습니다 — 즐겨찾기·인쇄해 두고 발표·강의 전에 한 번만 훑어보세요.

1. 공통 단축키 (모든 모드)

키동작활용
Space시작 / 일시정지발표 시작 큐, 인터벌 시작
R초기화시간 이상 발생 시 즉시 리셋
F전체화면 토글발표 시작 직전 1회
M모드 순환 (타이머 → 뽀모도로 → 스톱워치)한 세션에서 여러 모드 사용 시
S알림음 켜기/끄기 (타이머·뽀모도로) · 기록 저장 (스톱워치)조용한 환경에선 알림음 OFF
Esc전체화면 해제 / 모달 닫기발표 후 평상 모드 복귀

3. 뽀모도로 모드 전용

4. 스톱워치 모드 전용

키동작
L랩 기록
S기록 저장 (제목·메모 입력 모달)

5. 발표·강의용 시나리오 (마우스 0회)

시나리오 A: 18분 TED 발표

  1. 발표 직전 노트북에서 rstimer.kr/?t=18m 열기
  2. F 키로 풀스크린 진입 → 청중 시선이 무대로 집중
  3. 사회자 소개 후 Space 키로 18:00 카운트다운 시작
  4. 발표 진행 — 마지막 6초에 자동으로 화면 가득 카운트다운
  5. 완료 후 Esc로 풀스크린 해제 → Q&A 진행

시나리오 B: 학생 토론 5분 시간 측정

  1. 주제별 5분씩 분리 — 각 학생 차례에 R로 5:00 리셋, Space로 시작
  2. 학생이 마치면 R + 다음 학생 Space
  3. 모든 차례 끝 → 누적 사용 시간 자동 계산 안 됨 → 스톱워치 모드(M) 전환 후 측정

시나리오 C: 인터벌 운동 (HIIT)

  1. M으로 스톱워치 모드 → Space로 시작
  2. 각 라운드 끝에 L로 랩 → 랩 메모에 운동명 입력 (마우스 1회 — 메모 칸 클릭)
  3. 운동 종료 후 Space로 정지 → S로 저장 모달
  4. 제목 입력 (예: "타바타 버피 4/24") → 엔터 → 자동 저장
  5. 저장된 기록 카드의 "CSV 내보내기" 버튼 클릭 → 다운로드

6. 단축키 응용 팁

  • 발표 직전 손에 익히기: 발표 5분 전에 빈 슬라이드와 함께 ReadySetTimer를 한 번 돌려보세요. Space · F · Esc 시퀀스에 손을 익혀두면 무대에서 마우스 찾지 않습니다.
  • 한국어 입력기 끄고: 한글 입력 모드에서는 일부 단축키가 작동하지 않을 수 있습니다. 발표 시작 전 영문으로 전환.
  • 모달이 열린 상태에선 단축키 무효: 설정 모달이나 저장 모달이 열려 있을 때는 Space가 모달 안의 입력으로 들어갑니다. 모달을 Esc로 닫은 뒤 사용.
  • 모바일에선 단축키 미지원: 터치 기반 모바일은 단축키 없이 화면 버튼으로 조작합니다.
  • 설정에서 단축키 비활성화: 사용 환경에 따라 단축키를 끄고 싶다면 설정에서 토글할 수 있습니다 (향후 기능).

다음 텍스트를 노트북 옆에 붙여두면 발표 직전 빠르게 참고할 수 있습니다:
